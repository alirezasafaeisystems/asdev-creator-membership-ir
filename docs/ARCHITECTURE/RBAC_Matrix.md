# RBAC Matrix
نسخه: 1.1

## نقش‌ها
- `member`: دسترسی به محتوای خریداری‌شده و عملیات حساب کاربری.
- `creator`: مدیریت محتوای اختصاصی، درآمد و payoutهای خودش.
- `admin`: دسترسی مدیریتی به تنظیمات، گزارش‌ها و عملیات امنیتی.

## کنترل دسترسی
- تمام endpointهای `admin/*` فقط برای نقش `admin` مجاز هستند.
- عملیات حساس پرداخت/تسویه باید با `admin` یا مالک رکورد کنترل شوند.
- تغییر نقش کاربر فقط از مسیر ادمین و با ثبت رویداد ممیزی مجاز است.

## الزام ممیزی
- رویدادهای تغییر سطح دسترسی، ورود ناموفق تکراری و تغییرات policy باید log شوند.
